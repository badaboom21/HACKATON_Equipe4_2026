🧠 Présentation du projet

Ce repository contient le projet développé par l’équipe 4 dans le cadre du hackathon 2026.
L’objectif de ce projet est de concevoir une solution innovante répondant à une problématique donnée pendant l’événement, en combinant développement logiciel, créativité et travail collaboratif.

🚀 Objectifs

Développer une solution fonctionnelle en un temps limité
Mettre en pratique des compétences techniques (développement, UX, etc.)
Travailler en équipe de manière agile
Proposer un prototype innovant et présentable

🛠️ Technologies utilisées

Frontend : Angular
Backend : Java Spring Boot
Base de données : PostgreSQL
Autres outils : Git
